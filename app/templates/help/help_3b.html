Add pocket

A new pocket can be registered with name, a description and an initial balance.

Naming the new pocket is mandatory, provide description and initial balance is optional. In lack of initial balance, the balance will be 0.
To prevent inconsistency in the database, it is impossible to implicitly modify later the pocket balance.
At register the Blancer will create a default pocket wich is immediatelly ready to use, and can be renamed later.
